## 准备工作
一个 markdown 编辑器，我是比较推荐 markor 的，干净好用而且开源。  （同上图）
一个 termux 


## 部署
首先你需要在 termux 安装好`nodejs`    
然后安装 hexo-cli `npm i hexo-cli`
然后把整个 hexo 项目文件夹弄到手机上（我是直接丢 GitHub 然后clone下来。  
然后进入项目文件夹里面`npm i`（如果已经有 module 了的话，当我没说）  
就可以和电脑一样使用了。

## 编写
使用 markor 找到对应文件即可
